1. FUNGSI YANG TELAH DIIMPLEMENTASI
================================================================================

A. calculate_entropy(data)
   Lokasi: Line 106-134
   Fungsi: Menghitung entropy (tingkat keacakan) dataset
   Formula: Entropy(S) = -p_survived * log2(p_survived) - p_dead * log2(p_dead)
   
   Implementasi:
   - Menghitung jumlah yang selamat (survived = 1) dan meninggal (survived = 0)
   - Menghitung probabilitas masing-masing kelas
   - Mengembalikan 0 jika data homogen (semua sama)
   - Menggunakan math.log2() untuk perhitungan logaritma basis 2

B. calculate_information_gain(data, attribute_index)
   Lokasi: Line 137-167
   Fungsi: Menghitung information gain untuk setiap atribut
   Formula: Gain(S,A) = Entropy(S) - Weighted_Remainder_Entropy
   
   Implementasi:
   - Menghitung entropy total sebelum pemisahan
   - Memisahkan data berdasarkan nilai atribut
   - Menghitung weighted entropy untuk setiap subset
   - Mengembalikan selisih entropy (information gain)

C. build_tree(data, available_attributes)
   Lokasi: Line 170-225
   Fungsi: Membangun decision tree secara rekursif
   Algoritma: ID3 (Iterative Dichotomiser 3)
   
   Implementasi:
   - 3 Basis Kasus (Stopping Conditions):
     1. Data sudah homogen → return label
     2. Tidak ada atribut tersisa → return mayoritas
     3. Information gain <= 0 → return mayoritas
   
   - Proses Rekursif:
     1. Mencari atribut dengan information gain tertinggi
     2. Membuat node dengan atribut terbaik
     3. Memisahkan data berdasarkan nilai atribut
     4. Rekursif untuk setiap subset data

D. predict(tree, sample)
   Lokasi: Line 233-269
   Fungsi: Memprediksi hasil untuk data baru
   
   Implementasi:
   - Basis kasus: Jika sudah di leaf node, return nilai
   - Mengambil nama atribut dari kunci dictionary
   - Mencari indeks kolom dari attributes_map
   - Mengambil subtree berdasarkan nilai atribut
   - Rekursif sampai mencapai leaf node

- Pclass (Kelas Tiket): 1, 2, 3
- Age_Group (Kelompok Umur): 0 = Anak (≤16), 1 = Dewasa (17-50), 2 = Lansia (>50)

Struktur Decision Tree yang Dihasilkan:
- Root Node: Sex (Information Gain tertinggi)
- Level 2: Pclass
- Level 3: Age_Group (Leaf nodes)

Akurasi Model: 80.25%

3. INSIGHT DARI DECISION TREE
================================================================================

Pola yang Ditemukan:

A. Perempuan (Sex = 1):
   - Kelas 1 & 2: Semua umur SELAMAT (100% survival)
   - Kelas 3: Anak & Lansia SELAMAT, Dewasa MENINGGAL
   → "Women and children first" policy terlihat jelas

B. Laki-laki (Sex = 0):
   - Kelas 1 & 2: Hanya anak-anak yang SELAMAT
   - Kelas 3: Semua umur MENINGGAL
   → Prioritas penyelamatan sangat rendah untuk laki-laki

Kesimpulan:
1. Jenis kelamin adalah faktor terpenting (root node)
2. Kelas tiket menentukan akses ke sekoci penyelamatan
3. Usia berpengaruh terutama untuk prioritas anak-anak
